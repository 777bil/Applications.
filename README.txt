Campus chatbot

PROJECT OVERVIEW
This project demostrates the progressive development of a chatbot designed to answer common campus-related questions. It evolved from a rule-based model to a machine learning-based NLP classifier using TF-IDF and scikit-learn.

STAGE 1(RULE-BASED CHATBOT)
HOW IT WORKS
- User inputs text
- Input is checked for fixed keywords
- Keywords are matched using conditional rules
- Predefined responses are returned.

LIMITATIONS
- Cannot handle varied sentence structures.
- Not scalable

STGAE 2(MACHINE LEARNING-BASED CHATBOT)
HOW IT WORKS.
- User input is collected
- Text converted into numerical feaures(TF-IDF)
- ML model predicts user intent
- Chatbot returns corresponding response.

IMPROVEMENTS.
- Handles different phrasings
- Learns from data
